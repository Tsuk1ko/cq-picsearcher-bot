# CQ-picfinder-robot

这是一个以 Node.js 编写的酷Q机器人程序，用于搜图、搜番、搜本子，并夹带了许多娱乐向功能（。）

因酷Q已停止运营，本程序转为适配 [go-cqhttp](https://github.com/Mrs4s/go-cqhttp)

目前支持的搜图服务：

- [saucenao](https://saucenao.com)
- [WhatAnime](https://trace.moe)
- [ascii2d](https://ascii2d.net)

附加功能：

- 复读
- setu
- OCR
- 明日方舟公开招募计算
- 定时提醒
- 群发消息
- 反哔哩哔哩小程序

详细说明请移步 [Wiki](../../wiki)
